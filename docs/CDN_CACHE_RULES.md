# 腾讯云 CDN 缓存规则配置指南

## 当前状态

- ✅ 主域名 `vidluxe.com.cn` 已接入 CDN
- ✅ API 子域名 `api.vidluxe.com.cn` 不走 CDN（用于大文件上传）
- ⚠️ CDN 缓存规则需要优化

## 推荐的缓存规则

### 规则 1：静态资源 - 长期缓存

**路径规则**：`/_next/static/*`

**缓存配置**：
- 缓存时间：365 天（31536000 秒）
- 浏览器缓存：365 天
- 忽略查询参数：否
- 缓存键规则：完整 URL（包含查询参数）

**说明**：Next.js 的静态资源文件名包含哈希值，内容变化时文件名会变化，可以安全地长期缓存。

### 规则 2：上传文件 - 短期缓存

**路径规则**：`/uploads/*`

**缓存配置**：
- 缓存时间：1 天（86400 秒）
- 浏览器缓存：1 天
- 忽略查询参数：是

**说明**：用户上传的文件（图片、视频、关键帧）缓存 1 天，平衡性能和存储成本。

### 规则 3：API 请求 - 不缓存

**路径规则**：`/api/*`

**缓存配置**：
- 缓存时间：0 秒（不缓存）
- 浏览器缓存：0 秒

**说明**：API 请求返回动态数据，不应该被缓存。

### 规则 4：HTML 页面 - 不缓存

**路径规则**：
- `/`
- `/*.html`
- `/auth`
- `/try`
- `/pricing`
- `/dashboard`

**缓存配置**：
- 缓存时间：0 秒（不缓存）
- 浏览器缓存：0 秒

**说明**：HTML 页面包含对静态资源的引用，如果缓存会导致引用旧版本的资源文件（404 错误）。

### 规则 5：其他资源 - 默认缓存

**路径规则**：`/*`（默认规则，优先级最低）

**缓存配置**：
- 缓存时间：10 分钟（600 秒）
- 浏览器缓存：10 分钟

**说明**：其他未匹配的资源使用短期缓存。

## 配置步骤

### 1. 登录腾讯云控制台

访问：https://console.cloud.tencent.com/cdn

### 2. 选择域名

点击 `vidluxe.com.cn` → 缓存配置 → 缓存规则配置

### 3. 添加缓存规则

按照上述规则，从上到下依次添加（优先级从高到低）：

```
优先级 1: /_next/static/*     → 365 天
优先级 2: /uploads/*          → 1 天
优先级 3: /api/*              → 不缓存
优先级 4: /                   → 不缓存
优先级 4: /*.html             → 不缓存
优先级 4: /auth               → 不缓存
优先级 4: /try                → 不缓存
优先级 4: /pricing            → 不缓存
优先级 4: /dashboard          → 不缓存
优先级 5: /*                  → 10 分钟（默认）
```

### 4. 保存并刷新缓存

配置完成后，刷新 CDN 缓存：

1. 进入「缓存刷新」页面
2. 选择「目录刷新」
3. 输入：`https://vidluxe.com.cn/`
4. 提交刷新

## 验证方法

### 1. 检查静态资源缓存

```bash
curl -I https://vidluxe.com.cn/_next/static/css/xxx.css
```

**预期响应头**：

## 常见问题

### Q1: 部署后页面样式丢失

**原因**：CDN 缓存了旧的 HTML，引用了不存在的静态资源。

**解决**：
1. 确认 HTML 缓存规则为「不缓存」
2. 刷新 CDN 缓存（目录刷新：`/`）
3. 清除浏览器缓存（Cmd/Ctrl + Shift + R）

### Q2: 静态资源更新不生效

**原因**：CDN 缓存了旧版本的静态资源。

**解决**：
1. 刷新 CDN 缓存（目录刷新：`/_next/static/`）
2. 等待 5-10 分钟让 CDN 生效

### Q3: API 响应被缓存

**原因**：API 路径没有配置为「不缓存」。

**解决**：
1. 添加 `/api/*` 缓存规则，设置为不缓存
2. 刷新 CDN 缓存

## 监控指标

**目标**：> 90%

**目标**：< 10% 总带宽

**说明**：缓存命中率高，回源带宽就低，节省成本。

### 3. 响应时间

**目标**：
- 静态资源：< 100ms
- HTML 页面：< 500ms
- API 请求：< 2s
