# VidLuxe UI 设计方案

> 版本：1.0 | 日期：2026-02-17 | 状态：待确认

---

## 一、项目概述

### 1.1 产品定位
VidLuxe 是一款 AI 驱动的**高级感升级引擎**，帮助小红书中腰部博主将普通素材一键升级为杂志级质感内容。

### 1.2 核心价值
- **一键高级感**：普通素材 → 高级质感
- **AI 重构场景**：保留主体，重新生成惊艳背景
- **6 维度评分**：量化高级感提升效果

### 1.3 目标用户
- 小红书中腰部博主（粉丝 1-50 万）
- 内容创作者、自媒体运营
- 追求内容质感的品牌方

---

## 二、设计策略

### 2.1 设计理念
```
产品卖的是「高级感」→ UI 本身必须是高级感的最佳演示
```

### 2.2 核心洞察
| 用户痛点 | 我们的解决方案 |
|---------|--------------|
| 拍摄技术有限 | AI 自动重构专业级场景 |
| 后期耗时 | 一键升级，秒级出片 |
| 不知道高级在哪 | 6 维度可视化评分 |
| 不敢相信效果 | Before/After 直观对比 |

### 2.3 Before/After 核心逻辑
```
┌─────────────────────────────────────────────────┐
│  Before（左侧/展示区）   │   After（右侧/原片）   │
│                        │                        │
│  Nano Banana 生成      │   用户原始素材          │
│  高级场景 + 保留主体    │   普通光线/背景         │
│                        │                        │
│  "这是我能达到的效果！"  │  "这是我现在的问题"     │
└─────────────────────────────────────────────────┘
              拖动滑块 ← 一键高级感 → 哇效应
```

## 三、视觉风格

### 3.1 设计风格：Liquid Glass Premium
融合 Magnific AI 的功能布局 + Awwwards Premium 的视觉质感

### 3.2 色彩系统

```
主色调（Primary）
├── 金色主色：#CA8A04 / #D4AF37
├── 金色渐变：linear-gradient(135deg, #CA8A04, #EAB308)
└── 用途：CTA按钮、强调元素、滑块

背景色（Background）
├── 深黑主色：#0a0a0f
├── 深紫渐变：#0f0f18 → #0a0a0f
└── 用途：页面背景、卡片背景

辅助色（Accent）
├── 紫色光晕：rgba(139, 92, 246, 0.15)
├── 粉色光晕：rgba(236, 72, 153, 0.1)
├── 蓝色光晕：rgba(59, 130, 246, 0.1)
└── 用途：背景光斑、hover效果

文字色（Text）
├── 主文字：rgba(255, 255, 255, 0.9)
├── 次文字：rgba(255, 255, 255, 0.6)
├── 弱文字：rgba(255, 255, 255, 0.4)
└── 禁用文字：rgba(255, 255, 255, 0.2)
```

### 3.3 字体系统

```
主字体（Display）
├── Bodoni Moda
├── 用途：大标题、Logo
├── 特点：衬线体，优雅高级
└── Weight：400, 700

正文字体（Body）
├── Jost / Inter
├── 用途：正文、按钮、导航
├── 特点：无衬线，现代清晰
└── Weight：300, 400, 500, 600

字号规范
├── Hero 标题：48-72px / font-light
├── 区块标题：32-40px / font-medium
├── 正文：16-18px / font-normal
├── 辅助文字：14px / font-light
└── 标签：12px / tracking-wider
```

### 3.4 特效规范

```
玻璃效果（Glassmorphism）
├── backdrop-filter: blur(40px) saturate(150%)
├── background: rgba(20, 20, 30, 0.6)
├── border: 1px solid rgba(255,255,255,0.1)
└── box-shadow: 0 0 60px rgba(139,92,246,0.15)

光斑动画（Floating Orbs）
├── 圆形渐变：radial-gradient
├── 模糊：filter: blur(40-80px)
├── 动画：float 8-15s ease-in-out infinite
└── 透明度：0.1-0.3

滑块效果（Comparison Slider）
├── 滑块条：金色渐变 + 发光
├── 拖动手柄：圆形 + 放大hover
├── 分界线：1px solid rgba(255,255,255,0.3)
└── 过渡：transition: all 0.3s ease
```

**交互规范：**

- 分界线：1px white 30% opacity
- 手柄：圆形 40px，金色边框，白色背景
- 手柄图标：左右箭头 `< >`

### 7.2 交互动效

| 元素 | 触发 | 动效 |
|------|------|------|
| 卡片 | hover | scale(1.02) + 阴影增强 |
| 按钮 | hover | 亮度提升 + shimmer |
| 滑块 | drag | 实时响应 |
| 导航 | hover | 下划线展开 |
| 图片 | load | fade in |

### 7.3 滚动动画

```javascript
// 使用 Intersection Observer
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
    }
  });
}, { threshold: 0.1 });
```

## 十一、验收标准

### 11.1 视觉验收

- [ ] 整体风格符合 Liquid Glass Premium
- [ ] 色彩使用符合规范（金色主色 + 深黑背景）
- [ ] 字体使用正确（Bodoni Moda 标题 + Jost 正文）
- [ ] 玻璃效果正确实现
