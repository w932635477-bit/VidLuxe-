# 腾讯云 CDN 正确配置指南

## 重要说明

腾讯云 CDN 有两个不同的配置项：

### 1. 缓存键规则配置（Cache Key Rules）
- 位置：缓存配置 → 缓存键规则配置
- 作用：控制是否忽略 URL 参数和大小写
- 格式：
  ```
  类型: 全部文件/文件后缀/文件目录/文件路径/首页
  内容: 具体路径或后缀
  忽略参数: 不忽略/忽略全部/忽略指定参数
  忽略大小写: 是/否
  ```

### 2. 节点缓存过期配置（Node Cache Expiration）
- 位置：缓存配置 → 节点缓存过期配置
- 作用：控制 CDN 节点缓存文件的时间
- 格式：
  ```
  类型: 全部文件/文件后缀/文件目录/文件路径/首页
  内容: 具体路径或后缀
  缓存时间: 秒数
  ```

## 正确的配置步骤

### 第一步：配置节点缓存过期时间

访问：https://console.cloud.tencent.com/cdn/domains/vidluxe.com.cn?tab=cache

找到「节点缓存过期配置」，点击「新增规则」：

#### 规则 1：Next.js 静态资源（最重要）
```
类型: 文件目录
内容: /_next/static/
缓存时间: 31536000 秒（365天）
优先级: 1
```

#### 规则 2：用户上传文件
```
类型: 文件目录
内容: /uploads/
缓存时间: 86400 秒（1天）
优先级: 2
```

#### 规则 3：API 请求（不缓存）
```
类型: 文件目录
内容: /api/
缓存时间: 0 秒
优先级: 3
```

#### 规则 4：HTML 文件（不缓存）
```
类型: 文件后缀
内容: html
缓存时间: 0 秒
优先级: 4
```

#### 规则 5：首页（不缓存）
```
类型: 首页
内容: /
缓存时间: 0 秒
优先级: 5
```

#### 规则 6：关键页面（不缓存，需添加 4 次）
```
类型: 文件路径
内容: /auth
缓存时间: 0 秒
优先级: 6

类型: 文件路径
内容: /try
缓存时间: 0 秒
优先级: 7

类型: 文件路径
内容: /pricing
缓存时间: 0 秒
优先级: 8

类型: 文件路径
内容: /dashboard
缓存时间: 0 秒
优先级: 9
```

#### 规则 7：默认规则（最低优先级）
```
类型: 全部文件
内容: *
缓存时间: 600 秒（10分钟）
优先级: 99
```

### 第二步：配置缓存键规则（可选）

找到「缓存键规则配置」，保持默认即可：

```
类型: 全部文件
内容: 全部文件
忽略参数: 不忽略
忽略大小写: 否
```

如果需要优化，可以添加：

```
类型: 文件目录
内容: /_next/static/
忽略参数: 忽略全部
忽略大小写: 否
```

### 第三步：刷新 CDN 缓存

1. 进入「缓存刷新」页面
2. 选择「目录刷新」
3. 输入：
   ```
   https://vidluxe.com.cn/
   https://vidluxe.com.cn/_next/static/
   ```
4. 点击「提交」

### 第四步：验证配置（等待 5-10 分钟）

```bash
# 验证静态资源缓存（应该看到 Cache-Control: max-age=31536000）
curl -I https://vidluxe.com.cn/_next/static/css/app-layout.css

# 验证 HTML 不缓存（应该看到 Cache-Control: no-cache）
curl -I https://vidluxe.com.cn/

# 验证 API 不缓存（应该看到 Cache-Control: no-cache）
curl -I https://vidluxe.com.cn/api/health
```
