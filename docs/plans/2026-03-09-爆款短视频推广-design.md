## 三、具体操作方法

### 3.1 如何找爆款模板

**搜索关键词**（抖音/小红书）：
```
修图对比、原图vs精修、照片升级
素人改造、一键变美、ai修图
拍照技巧、博主日常、神转折
```

**筛选标准**：
- 点赞 1 万+（已验证有效）
- 近 1 个月发布（不过时）
- 结构简单（容易模仿）
- 有植入空间（能加产品）

**记录模板**：
```
视频链接：
开头（前3秒）：
中间铺垫：
转折/高潮：
结尾/行动号召：
可以植入的点：
```

### 3.2 产品植入方式

| 植入方式 | 示例 | 自然度 |
|---------|------|--------|
| **对比展示** | "左边原图，右边用 VidLuxe 升级后" | ⭐⭐⭐⭐⭐ |
| **问题解决** | "照片不够高级？试试这个工具" | ⭐⭐⭐⭐ |
| **揭秘分享** | "博主不告诉你的秘密..." | ⭐⭐⭐⭐⭐ |
| **搞笑植入** | "女王为什么爱上唐僧？因为..." | ⭐⭐⭐⭐ |

**植入原则**：
- 产品是故事的一部分，不是硬广
- 先给价值/娱乐，再软性植入
- 结尾给出行动指引（搜索 vidluxe.com.cn）

### 3.3 视频制作工具

| 工具 | 用途 | 费用 |
|------|------|------|
| **剪映** | 剪辑、字幕、特效 | 免费 |
| **CapCut** | 剪映国际版，模板多 | 免费 |
| **Canva** | 封面图设计 | 免费 |

**素材来源**：
- 产品截图：自己录制
- 对比图：用 VidLuxe 生成
- 背景音乐：剪映自带

### 3.4 发布技巧

**最佳发布时间**：

| 平台 | 时间段 |
|------|--------|
| 抖音 | 12:00-13:00, 18:00-21:00 |
| 小红书 | 8:30, 12:30, 20:00 |

**标题公式**：
```
"一张照片，两个价格感"
"我悟了！原来高级感就差这一步"
"博主不告诉你的修图秘密"
```

**话题标签**：
```
#AI修图 #高级感 #照片升级 #修图技巧
#博主日常 #小红书运营 #穿搭
```
