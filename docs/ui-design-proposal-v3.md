## 一、项目概述

### 1.1 产品定位
VidLuxe 是一款 AI 驱动的**高级感升级引擎**，帮助小红书中腰部博主将普通素材一键升级为杂志级质感内容。

### 1.2 核心能力
| 能力 | 说明 | MVP 状态 |
|------|------|---------|
| 图片升级 | AI 重构场景 + 保留主体 | ✅ 优先 |
| 视频升级 | 帧级分析 + 批量处理 | 🔜 二期 |
| 4 维评分 | 量化高级感提升 | ✅ 核心 |
| 4 种风格 | 极简/暖调/冷调/莫兰迪 | ✅ 核心 |

### 1.3 目标用户画像

**核心用户：小红书中腰部博主**
- 粉丝：1-50 万
- 内容：穿搭/美妆/生活方式/探店
- 痛点：拍摄技术有限、后期耗时、缺乏量化标准
- 场景：早上拍素材 → 晚上 VidLuxe 升级 → 第二天发布
- 设备：主要用手机，偶尔用电脑
