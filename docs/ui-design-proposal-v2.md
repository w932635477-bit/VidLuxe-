## 一、项目概述

### 1.1 产品定位
VidLuxe 是一款 AI 驱动的**高级感升级引擎**，帮助小红书中腰部博主将普通素材一键升级为杂志级质感内容。

### 1.2 核心能力
| 能力 | 说明 | MVP 状态 |
|------|------|---------|
| 图片升级 | AI 重构场景 + 保留主体 | ✅ 优先 |
| 视频升级 | 帧级分析 + 批量处理 | 🔜 二期 |
| 6 维评分 | 量化高级感提升 | ✅ 核心 |
| 4 种风格 | 极简/暖调/冷调/莫兰迪 | ✅ 核心 |

### 1.3 目标用户画像

**核心用户：小红书中腰部博主**
```
┌─────────────────────────────────────────────────────┐
│  小红书博主 @某某                                    │
│                                                     │
│  粉丝：1-50 万                                      │
│  内容：穿搭/美妆/生活方式/探店                        │
│  痛点：                                             │
│  • 拍摄技术有限，成片不够高级                         │
│  • 后期耗时，每张图要修 30 分钟+                      │
│  • 不知道高级在哪，缺乏量化标准                       │
│  • 竞品内容越来越卷，需要提升质感                     │
│                                                     │
│  场景：                                             │
│  • 早上拍素材 → 晚上 VidLuxe 升级 → 第二天发布        │
│  • 主要用手机，偶尔用电脑                             │
│  • 追求效率，希望一键出片                             │
└─────────────────────────────────────────────────────┘
```

### 1.4 差异化价值

| VidLuxe | 竞品（如 Magnific AI） |
|---------|----------------------|
| 中文界面，小红书场景 | 英文界面，通用场景 |
| 6 维度评分系统 | 无评分 |
| 4 种高级感风格 | 单一风格 |
| 9:16 竖版优化 | 16:9 横版 |
| Nano Banana 场景重构 | 简单增强 |
